# GitHub Upload Pack — Dummy Structured SFT Generators

This folder documents what to include when publishing the synthetic data generators to GitHub.

Include the following paths from this repository:

- `tools/generate_dummy_structured_data.py`
- `tools/generate_hard_structured_data.py`
- `tools/README.md`
- `/LICENSE` (Apache‑2.0)

Optional references (not required for generation but useful to link):

- `scripts/train_local_qlora_qwen3.py` — example local trainer consuming JSONL messages

Repository landing `README.md` can briefly point to `tools/README.md` for full script docs.
